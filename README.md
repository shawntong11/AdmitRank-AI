# AdmitRank AI — Graduate School Admissions Analyzer

🚀 **[Try it Live → shawntong11.github.io/AdmitRank-AI](https://shawntong11.github.io/AdmitRank-AI)**

AdmitRank AI is an open-source, AI-powered admissions diagnostic tool. Upload your resume PDF → get a full admissions analysis in 30 seconds. No sign-up required. Powered by Claude AI.

If this helped you, please ⭐ **star this repo** — it helps others find it!

---

## What You Get

- 📋 **1. Admissions Competitiveness Report** — Overall score, dimension breakdown, strengths & weaknesses
- 🏛️ **2. School Selection Strategy** — 3-tier school matching with personalized acceptance probabilities
- 🔬 **3. Advisor Matching** — Faculty recommendations based on your research interests
- 🚀 **4. Improvement Tips** — Actionable steps to strengthen your application
- 🗓️ **5. Application Timeline** — Month-by-month roadmap to submission

---

## Quick Start

1. **[Open the tool](https://shawntong11.github.io/AdmitRank-AI)** in your browser
2. Enter your [Anthropic API Key](https://console.anthropic.com/keys) *(stored locally, never uploaded)*
3. Upload your resume PDF
4. Click **Analyze** → results in ~30 seconds

> **Don't have an API Key?** Sign up at [console.anthropic.com](https://console.anthropic.com) — new accounts get free credits.

## Run Locally

No installation needed. Just download and open:

```bash
git clone https://github.com/shawntong11/AdmitRank-AI.git
cd AdmitRank-AI
open index.html
```
